I’m going to start posting about cool handy Chrome plugins I find. One of my favorite ones right now is Wappalyzer. This handy plugin detects what technologies a website is using. 


It detects CMS, Javascript frameworks, Analytics, Programming language and even the server! Long are the days where you dig into the source trying to find this info.

It’s also available for Firefox.
